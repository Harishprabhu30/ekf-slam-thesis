# Hybrid AI-Augmented EKF-SLAM System for Robust Indoor Navigation

## Repository Structure
- `src/ekf_slam_sim/` - Main ROS2 package with all nodes
- `docs/` - Thesis documentation (optional)
- `scripts/` - Setup and utility scripts
- `config/` - Configuration files for different scenarios

## Setup Instructions
1. Clone repository
2. Build with `colcon build`
3. Launch simulation with `./setup_sim.sh`
